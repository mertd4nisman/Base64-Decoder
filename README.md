# Fast Base64 Decoder (Python)


[TR] Bu proje, terminal üzerinden Base64 kodlu metinleri anında ve güvenli bir şekilde orijinal haline geri dönüştüren (decode) hafif ve pratik bir Python script'idir.

[EN] This project is a lightweight and practical Python script that instantly decodes Base64 encoded strings back to their original form via the terminal.

---

## ✨ Özellikler / Features

- ⚡ **Hızlı ve Hafif (Lightweight):** Ekstra hiçbir kütüphane (`pip install`) gerektirmez, Python'ın yerleşik `base64` modülünü kullanır.
- 🔁 **Sürekli Döngü (Interactive CLI):** Programı kapatmadan üst üste birden fazla kodu çözebilirsiniz.
- 🛡️ **Hata Yakalama (Error Handling):** Hatalı veya eksik (incorrect padding) Base64 kodu girildiğinde program çökmez, sizi uyarır ve çalışmaya devam eder.

---

## 🛠️ Kurulum ve Çalıştırma / Installation & Usage

Projenin bilgisayarınızda çalışması için sadece **Python 3.x** kurulu olması yeterlidir.
